### Step 2: Generate QR Codes

1. **Double-click** `qr-generator.html` (opens in browser)
2. **Paste your URL** from Step 1 into the input box
3. **Click download** for each of the 5 QR codes:
   - Harmony Haven
   - Wisdom Well
   - Peace Pavilion
   - Serenity Square
   - Bliss Bay
4. **Save all 5 images** to your computer

### Step 3: Print QR Codes

1. **Open each QR code image**
2. **Print settings:**
   - Paper size: A4 or Letter (8.5" √ó 11")
   - Quality: Best/High
   - Color: Color (not black & white)

### Step 4: Set Up at Event

**For Each Stall:**
1. Place QR code poster at entrance
2. Mount at eye level (about 5-6 feet high)
3. Ensure good lighting
4. Test scan with your phone

**Admin Setup:**
1. Give each stall admin the web app URL
2. Share Admin PIN: `1234` (or your custom PIN)
3. Show them how to:
   - Open app in browser
   - Go to "Admin" tab
   - Enter PIN
   - Select their stall
   - Award points to players

---

### Step 5: Test Before Event

1. **Scan a QR code** with your phone
2. **Allow camera permission** when asked
3. Check that you get a random animal name
4. **Log in as admin:**
   - Go to Admin tab
   - Enter PIN: `1234`
   - Select a stall
   - Award yourself test points
5. **Check leaderboard** - you should see your name!

## üÜò Emergency Troubleshooting

**QR Code won't scan:**
- Is there good lighting?
- Hold phone steady for 2-3 seconds
- Try different angle
- Make sure camera is allowed in browser

**Admin can't log in:**
- Check PIN is correct: `1234`
- Make sure you're on Admin tab
- Refresh the page

**Player not in admin list:**
- Player must scan a QR code first
- Refresh admin page
- Check you selected correct stall

**Need to change Admin PIN:**
- Send message to your technical support person
- They can update it in the code
